# Lernperiode-13

## Entscheidung

Ich habe mich dazu entschieden, eine kleine Lifecounter App für SWU zu erstellen. Da ich eine neue Programmiersprache lernen möchte, welche darauf ausgelegt ist plattfromübergreifende Apps zu programmieren wähle ich Flutter (Dart).

### Flutter (Dart)
- **Pros:**
  - Plattformübergreifendes UI-Framework
  - Reiche Widgets und schnelle UI-Entwicklung
  - Große Community und Ökosystem
- **Cons:**
  - Größere Binärgrößen
  - Performance bei Desktop noch nicht immer optimal
  - Dart ist weniger verbreitet

### Tauri
- **Pros:**
  - Sehr schlanke Anwendungen
  - Hohe Sicherheit
  - Rust-basiertes Backend
- **Cons:**
  - Keine rein native UI-Entwicklung
  - Kleinere Community
